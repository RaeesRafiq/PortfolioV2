A high-performance, modern portfolio built with **React 19** and **Vite**, featuring a brutalist-inspired dark aesthetic, smooth animations, and a seamless user experience.

---

## ✨ Features

- 🏎️ **Vite 8 Powered**: Lightning-fast Hot Module Replacement (HMR) and optimized builds.
- 🎨 **Modern Brutalist Design**: Striking aesthetic featuring fully functional **Light** and **Dark** modes via a smooth toggle.
- 🎭 **Framer Motion Animations**: Smooth, high-quality transitions, interactive components, and scroll-reveal effects.
- 🖱️ **Custom Cursor Engine**: Interactive lens effect cursor that reacts to the interface.
- 📜 **Lenis Smooth Scroll**: Premium momentum scrolling experience across all browsers.
- 📊 **GitHub Pulse**: Live integration of GitHub contribution activity.
- 📱 **Fully Responsive Layouts**: Optimized and perfectly centered for all viewports, featuring a seamless animated mobile dropdown menu and dedicated ultra-wide screen constraints.
- 🛠️ **Tailwind CSS 4**: Utilizing the latest in utility-first CSS for efficient and modular styling.

---

## 🚀 Tech Stack

- **Core**: React 19, Vite 8
- **Styling**: Tailwind CSS 4, Custom CSS variables for theme management
- **Animations**: Framer Motion
- **Performance**: Lenis (Smooth Scroll), @tailwindcss/vite
- **Data**: react-github-calendar

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/PortfolioV2.git
   cd PortfolioV2/Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
